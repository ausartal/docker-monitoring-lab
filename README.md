\# Docker Monitoring Lab



Mini DevOps monitoring stack using:



\- Docker Compose

\- NGINX

\- PostgreSQL

\- Prometheus

\- Grafana

\- cAdvisor

\- Keycloak



\## Features



\- Container monitoring

\- Realtime metrics

\- Grafana dashboards

\- PostgreSQL database

\- Authentication with Keycloak

\- Docker observability stack



\## Services



| Service | Port |

|---|---|

| NGINX | 8080 |

| PostgreSQL | 5432 |

| Prometheus | 9090 |

| Grafana | 3000 |

| cAdvisor | 8088 |

| Keycloak | 8081 |



\## Run



```bash

docker compose up -d
